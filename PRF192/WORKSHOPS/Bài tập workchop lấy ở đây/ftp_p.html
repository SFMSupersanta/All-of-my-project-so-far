File Transfer Protocol



Local to remote and remote to local

File transfer protocol (ftp) is an industry standard protocol available on all computers that run Windows, UNIX and Linux to transfer files between computers. 

We can access ftp through a graphical open-source program like WinSCP or through a browser like Internet Explorer. 

WinSCP

You can download a copy of WinSCP here.

Browser

In the browser's address bar, type 


 ftp://userid@phobos.senecac.on.ca

and press the GO button.  userid is the remote computer's userid and phobos is the name of the remote computer.  The ftp will prompt you for your password and, once the remote computer has logged you in, will display the contents of the remote computer's home directory in the browser window. 

Open a second browser window and navigate to the directory on your local computer.  For example, 


 c:\temp

and press the GO button. 

Downloading

In the remote computer window, select the icon of the file to download and drag and drop the icon into the local computer window. 

Uploading

To upload a file from the local computer, login through the address bar (as above), select the icon of the file to upload in the local computer window and drag and drop the icon into the remote computer's window.
