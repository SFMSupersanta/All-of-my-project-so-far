Assignment 6

Income Statement and Balance Sheet with Files




As your sixth and final assignment this semester, you are to amend your bookkeeping program to store

  • the accounts in a sequential text file named accounts.dat
  • the transactions in a sequential text file named transactions.dat

If these file(s) exist with account and/or transaction information, then that information will be read and used by your program to populate the relevant arrays before the user is asked for input


Learning Outcome

Upon successful completion of this assignment, you will have demonstrated the ability to use sequential text files.


Specifications

This version of your bookkeeping program stores all account information in a file named accounts.dat  The accounts.dat file contains one transaction per record and each record should contain two fields delimited by the colon character ':'.  The two fields are:

  • account number
  • account title
For example, a few records might look like: 

 1000:Cash
 3000:Common Shares Outstanding
 1400:Office Equipment
 2000:Credit Card Payable
 2100:Long Term Debt

Your program limits the number of accounts to 100.

Also, this version of your bookkeeping program stores all transaction information in a file named transactions.dat  The transaction file contains one transaction per record and each record should contain four fields delimited by the colon character ':'.  The four fields are:

  • account number
  • transaction title
  • transaction type
  • transaction amount
For example, a few records might look like: 

 1000:Common Shares Issued:d:4000.00
 3000:Investor #2314:c:4000.00
 1400:Office Equipment:d:2100.99
 2000:Future Shop:c:2100.99
 2100:Royal Bank:c:3500.00
 1000:Long Term Loan:d:3500.00

Your program limits the number of transactions to 20.

Whenever the user enters a new transaction, your program stores the transaction information at the end of the transactions.dat file.  If the transaction introduces a new account, your program stores the account information at the end of the accounts.datfile.
