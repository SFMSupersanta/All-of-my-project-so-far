Practice

validName



Write a function with the header 

 int validName ( char name[ ] )

Your function receives a null-terminated string name[] that contains a variable, array or function name.  Your function returns a true value if name[] contains a valid variable, array or function name, a false value if not.

A variable, array or function name is valid if it contains letters, digits and underscores in any order, but the first character is not a digit and the name is not a keyword.  Keywords are stored as newline-delimited records in a text file named keywords.txt  Each record contains a single keyword.

For example, the calls

validName("if") returns a false value

validName("ii") returns a true value
