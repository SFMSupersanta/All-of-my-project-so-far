Conversions

Workshop 2


In this workshop, you are to experiment with the double floating-point data type.


Learning Outcome

Upon successful completion of this workshop, you will be able to discuss different aspects of how data is stored in the C double data type and converted to the C int type.


Adding Doubles

Enter, compile and run the following program for the sample data noted.  

 /* Working with Doubles - Adding Doubles 
 *  BTP100 - Workshop 2
 *  Week 2
 *  adding.c
 *
 *  sample data: 123.00 => 123.09
 */

 main() {

     double price, gst, pst, total;

     printf("Enter the price : ");
     scanf("%lf", &price);

     gst = 0.08 * price;
     pst = 0.07 * price;
     total = price + gst +  pst;

     printf("Price : $%7.2lf\n", price);
     printf("GST   : $%7.2lf\n", gst);
     printf("PST   : $%7.2lf\n", pst);
     printf("Total : $%7.2lf\n", total);

     return 0;
 }

Then, do the following:

  • Record the output values for each test case.
  • Note the cases where the addition appears to be incorrect. 
  • Guess the source of the inaccuracies. 
  • Temporarily change the conversion specifiers to %7.3lf to confirm the source of the inaccuracies, then change them back.
  • Modify the code to ensure correct addition in all of the test cases.

Rounding Doubles

Enter, compile and run the following program with the sample data noted. 

 /* Working with Doubles - Rounding Doubles
 *  BTP100 - Workshop 2
 *  Week 2
 *  rounding.c
 *
 *  test data: 12.3, 12.7, -12.3, -12.7
 */

 main() {

     double value, remainder;
     int truncated, rounded;

     printf("Enter a value : ");
     scanf("%lf", &value);

     truncated = value;
     remainder = value - truncated;
     rounded = value;

     printf("Value     : %10.2lf\n", value);
     printf("Truncated : %7d\n", truncated);
     printf("Remainder : %10.2lf\n", remainder); 
     printf("Rounded   : %7d\n", rounded);

     return 0;
 }

Then, do the following:

  • Record the output values for each test case.
  • Note the cases where the rounding appears incorrect. 
  • Rewrite the rounded = assignment statement to yield the correct result for positive value. 
  • Add code to yield the correct result for both positive and negative values. 

Input and Output using printf and scanf (Challenge)

Enter, compile and run the following program with the sample data noted. 

 /* Working with Doubles - Rounding with printf and scanf 
 *  BTP100 - Workshop 2
 *  Week 2
 *  printf_scanf.c
 *
 *  sample data: 1.999, 1.999999999, 1.9999999999999999
 */

 main() {

     double value;

     printf("Enter a value : ");
     scanf("%lf", &value);

     printf("The value accepted was in 20.16 format"
            " : %20.16lf\n", value);
     printf("The value accepted was in 14.10 format"
            " : %14.10lf\n", value);
     printf("The value accepted was in  6.2  format"
            " : %6.2lf\n", value);

     return 0;
 }

Then, do the following:

  • Tabulate the input values, your guess of the contents of memory and the output values. 
  • Note the cases where the output appears incorrect. 
  • Explain the results that you observe for each test case.
