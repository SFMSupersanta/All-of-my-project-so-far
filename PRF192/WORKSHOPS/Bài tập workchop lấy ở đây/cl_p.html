Visual Studio .net 2003 Compiler



The Visual Studio .net 2003 compiler is a command line version of the same compiler that ships with Visual Studio .net 2003 Professional.  The command line compiler is available for free and can be downloaded from the Microsoft Development Center.  The Toolkit includes the core tools needed to compile and run source code on a Windows platform, such as the C runtime library.  To install the compiler, execute the download and follow the instructions. 

Once you have installed the compiler, you can access it by opening a command window from the Start menu at All Programs/Microsoft Visual C++ Toolkit 2003/Visual C++ Toolkit Command Prompt and typing 


 cl /?

at the command line prompt.  This command will list all of the compiler options. 

To compile a file named myFile.c follow the cl command by the source file name 


 cl myFile.c

If your source code compiles successfully, this will produce an executable file named myFile.exe.  To execute this file, enter 

 myFile

Note that the command window for the cl compiler differs from the normal command window.
