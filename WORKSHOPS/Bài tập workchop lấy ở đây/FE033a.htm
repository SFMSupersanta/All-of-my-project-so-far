Question 1 (10 marks):

Design and code a function with the header 

int justComments(char filename[ ])
that displays all of the comments in a C source file named filename[ ].  Each comment should appear on a separate line and start in the first column of that line.  Your function should display multi-line comments as they appear in the source file, but with the first line starting in the first column.  Your function should return 0 if successful, 1 if the file is not found.  You may assume that there are no errors while reading the file. 

For example, if the file named source.c consists of the following statements 

/* Sample File
*  source.c
*/

int main(void){
	int i;
	char s[100];

	/* prompt for input */
	printf("Enter a string : ");
	/* accept a string */
	scanf("%s", s);
	/* display the string entered */
	printf("%s\n", s);

	return 0;
}
then the statements 
int i;
i = justComments("source.c");
should display 
/* Sample File
*  source.c
*/
/* prompt for input */
/* accept a string */
/* display the string entered */
and assign a value of 0 to i.  The following statements 
int i;
i = justComments("");
should assign a value of 1 to i.

(Bonus 2 marks - if you have time) Enhance your function so that each comment displays in the same columns as the comment appears in the source file.  That is, if a comment starts in column 8 and ends in column 16 in the source file, display the comment in columns 8 through 16 inclusive.  You may assume that each tab character in the source file expands to 8 spaces on display.

Question 3 (10 marks):

What is the exact output of the following program?  Show your rough work to avoid deductions. 

#include <stdio.h>
#define N 8
void r ( int a[], int n );

int main ( void ) {
	int i, j, a[N];

	r ( a, N );
	for ( i = 0; i < N; i++ ){
		printf("||");
		for ( j = 0; j < a[i]; j++ )
			printf("-");
		printf("\n");
	}
	return 0;
}

void r ( int a[], int n ) {
	int i, x;

	for ( i = 0; i < n; i++ ){
		x = i + 1;
		a[i] = 1;
		while ( x % 2 == 0 ) {
			a[i]++;
			x /= 2;
		}
	}
}

	for ( i = 0; s[i] != '\0'; i++ ) {

	}
